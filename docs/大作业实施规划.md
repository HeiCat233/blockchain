# 基于区块链技术的分布式数字身份系统大作业实施规划

> **版本说明**：本规划已迁移至**长安链（ChainMaker）**作为区块链底层技术平台，并进行了全面的架构优化。

---

## 一、项目概述

### 1.1 项目背景与意义

**分布式数字身份（DID）**是基于去中心化标识符的身份管理新范式，通过区块链技术实现用户对自己身份数据的自主控制。

**核心价值：**
- 🔐 **数据主权**：用户拥有并控制自己的身份数据
- 🛡️ **隐私保护**：选择性披露，最小化信息泄露
- 🔗 **互操作性**：跨平台、跨系统身份通用
- ✅ **可信验证**：区块链提供不可篡改的身份记录

### 1.2 项目目标

**主要目标：**
1. 构建基于长安链的分布式数字身份基础设施
2. 实现身份的注册、验证、更新、吊销全生命周期管理
3. 提供安全、高效的身份验证服务
4. 实现多方参与的身份管理机制

**技术指标：**

| 指标项 | 目标值 |
|--------|--------|
| 系统并发处理能力 | ≥1000 TPS |
| 身份验证响应时间 | < 2秒 |
| 数据可用性 | ≥ 99.9% |
| 安全等级 | 等保2.0标准 |

## 二、技术选型分析

### 2.1 为什么选择长安链

经过技术选型分析，本项目选择**长安链（ChainMaker）**作为区块链底层平台，相比Hyperledger Fabric具有以下优势：

| 对比维度 | Hyperledger Fabric | 长安链 |
|----------|-------------------|--------|
| **开发语言** | Go/Java/Node.js | Go |
| **国密支持** | 需额外配置 | 原生支持 |
| **文档语言** | 英文为主 | 中文文档完善 |
| **学习曲线** | 陡峭 | 平缓 |
| **部署复杂度** | 较高 | 简化 |
| **社区支持** | 国际社区 | 国内生态 |
| **合规适配** | 需适配 | 易于合规 |

**长安链核心优势：**
- 🚀 **性能领先**：交易吞吐能力达10万+TPS
- 🔐 **国密原生**：内置SM2/SM3/SM4国密算法
- 📝 **中文文档**：开发文档详尽，降低学习成本
- 🔧 **灵活装配**：模块化设计，支持按需定制
- 🛡️ **隐私保护**：支持零知识证明、同态加密

### 2.2 迁移策略

本项目从Hyperledger Fabric迁移至长安链，遵循以下策略：

1. **业务逻辑不变**：保持原有的身份注册、验证、更新、吊销等功能
2. **架构优化**：利用长安链的特性进行架构简化
3. **渐进式迁移**：分模块逐步迁移，降低风险
4. **向后兼容**：保留Hyperledger Fabric的代码结构，便于对比学习

## 五、长安链SDK封装详解

### 5.1 SDK客户端封装

为了简化与长安链的交互，我们封装了统一的SDK客户端。

```typescript
// backend/src/chainmaker/client.ts

import * as ChainMakerSDK from 'chainmaker-sdk-js';
import * as fs from 'fs';
import * as path from 'path';

/**
 * 长安链SDK客户端封装
 * 提供统一的区块链交互接口
 */
export class ChainMakerClient {
  private sdk: any;
  private chainId: string;
  private orgId: string;
  private userSignFile: string;
  private chainCertFile: string;
  
  constructor(config: ChainMakerConfig) {
    this.chainId = config.chainId;
    this.orgId = config.orgId;
    this.userSignFile = config.userSignFile;
    this.chainCertFile = config.chainCertFile;
  }
  
  /**
   * 初始化SDK连接
   */
  async connect(): Promise<void> {
    const { ChainMaker } = ChainMakerSDK;
    
    // 读取证书文件
    const userSignBytes = fs.readFileSync(this.userSignFile);
    const chainCertBytes = fs.readFileSync(this.chainCertFile);
    
    // 创建SDK配置
    const sdkConfig = {
      chainId: this.chainId,
      orgId: this.orgId,
      userSignKeyBytes: userSignBytes,
      userSignCertBytes: userSignBytes,
      chainNodes: [
        {
          nodeAddr: '127.0.0.1:12301',
          tlsCaCertBytes: chainCertBytes
        }
      ]
    };
    
    this.sdk = await ChainMaker.newInstance(sdkConfig);
    console.log(`[ChainMaker] Connected to chain ${this.chainId}`);
  }
  
  /**
   * 调用智能合约（发送交易）
   * @param contractName 合约名称
   * @param method 合约方法名
   * @param params 参数列表
   */
  async invokeContract(
    contractName: string, 
    method: string, 
    params: Map<string, string>
  ): Promise<InvokeResult> {
    try {
      // 序列化参数
      const paramBytes = this.serializeParams(params);
      
      // 发送交易
      const result = await this.sdk.transaction.invokeContract({
        contractName,
        method,
        paramBytes,
        syncResult: true  // 同步等待结果
      });
      
      return {
        success: result.code === 0,
        txId: result.txId,
        message: result.message,
        data: result.result
      };
    } catch (error: any) {
      console.error(`[ChainMaker] Invoke failed: ${error.message}`);
      return {
        success: false,
        txId: '',
        message: error.message
      };
    }
  }
  
  /**
   * 查询智能合约（不发送交易）
   * @param contractName 合约名称
   * @param method 合约方法名
   * @param params 参数列表
   */
  async queryContract(
    contractName: string, 
    method: string, 
    params: Map<string, string>
  ): Promise<QueryResult> {
    try {
      const paramBytes = this.serializeParams(params);
      
      const result = await this.sdk.transaction.queryContract({
        contractName,
        method,
        paramBytes
      });
      
      return {
        success: true,
        data: result.result
      };
    } catch (error: any) {
      console.error(`[ChainMaker] Query failed: ${error.message}`);
      return {
        success: false,
        error: error.message
      };
    }
  }
  
  /**
   * 序列化参数为字节数组
   */
  private serializeParams(params: Map<string, string>): Buffer {
    const result: string[] = [];
    params.forEach((value, key) => {
      result.push(key, value);
    });
    return Buffer.from(result.join(','));
  }
  
  /**
   * 断开连接
   */
  async disconnect(): Promise<void> {
    if (this.sdk) {
      await this.sdk.close();
      console.log('[ChainMaker] Disconnected');
    }
  }
}

interface ChainMakerConfig {
  chainId: string;
  orgId: string;
  userSignFile: string;
  chainCertFile: string;
}

interface InvokeResult {
  success: boolean;
  txId: string;
  message?: string;
  data?: any;
}

interface QueryResult {
  success: boolean;
  data?: any;
  error?: string;
}
```

### 5.2 智能合约调用封装

```typescript
// backend/src/chainmaker/contract.ts

import { ChainMakerClient } from './client';

/**
 * 身份管理合约调用器
 * 封装所有与身份相关的区块链操作
 */
export class IdentityContract {
  private client: ChainMakerClient;
  private contractName: string = 'identity_contract';
  
  constructor(client: ChainMakerClient) {
    this.client = client;
  }
  
  /**
   * 注册新的数字身份
   * @param did 去中心化标识符
   * @param didDocument DID文档
   */
  async register(did: string, didDocument: string): Promise<InvokeResult> {
    const params = new Map([
      ['did', did],
      ['did_document', didDocument]
    ]);
    
    return await this.client.invokeContract(
      this.contractName,
      'RegisterIdentity',
      params
    );
  }
  
  /**
   * 验证身份有效性
   * @param did 去中心化标识符
   */
  async verify(did: string): Promise<boolean> {
    const params = new Map([['did', did]]);
    
    const result = await this.client.queryContract(
      this.contractName,
      'VerifyIdentity',
      params
    );
    
    return result.success && result.data === 'true';
  }
  
  /**
   * 查询身份详情
   * @param did 去中心化标识符
   */
  async query(did: string): Promise<string | null> {
    const params = new Map([['did', did]]);
    
    const result = await this.client.queryContract(
      this.contractName,
      'QueryIdentity',
      params
    );
    
    if (result.success && result.data) {
      return result.data;
    }
    return null;
  }
  
  /**
   * 更新身份信息
   * @param did 去中心化标识符
   * @param newDidDocument 新的DID文档
   */
  async update(did: string, newDidDocument: string): Promise<InvokeResult> {
    const params = new Map([
      ['did', did],
      ['did_document', newDidDocument]
    ]);
    
    return await this.client.invokeContract(
      this.contractName,
      'UpdateIdentity',
      params
    );
  }
  
  /**
   * 吊销身份
   * @param did 去中心化标识符
   */
  async revoke(did: string): Promise<InvokeResult> {
    const params = new Map([['did', did]]);
    
    return await this.client.invokeContract(
      this.contractName,
      'RevokeIdentity',
      params
    );
  }
  
  /**
   * 查询身份历史记录
   * @param did 去中心化标识符
   */
  async queryHistory(did: string): Promise<any[]> {
    const params = new Map([['did', did]]);
    
    const result = await this.client.queryContract(
      this.contractName,
      'QueryIdentityHistory',
      params
    );
    
    if (result.success && result.data) {
      return JSON.parse(result.data);
    }
    return [];
  }
}

### 5.3 交易管理封装

```typescript
// backend/src/chainmaker/transaction.ts

import { ChainMakerClient } from './client';

/**
 * 交易管理服务
 * 提供交易状态查询、重试等高级功能
 */
export class TransactionManager {
  private client: ChainMakerClient;
  private maxRetries: number = 3;
  private retryDelay: number = 1000; // 毫秒
  
  constructor(client: ChainMakerClient) {
    this.client = client;
  }
  
  /**
   * 发送带重试的交易
   * @param contractName 合约名称
   * @param method 方法名
   * @param params 参数
   * @param options 重试配置
   */
  async invokeWithRetry(
    contractName: string,
    method: string,
    params: Map<string, string>,
    options: RetryOptions = {}
  ): Promise<InvokeResult> {
    const maxRetries = options.maxRetries ?? this.maxRetries;
    const retryDelay = options.retryDelay ?? this.retryDelay;
    
    let lastError: Error | null = null;
    
    for (let attempt = 1; attempt <= maxRetries; attempt++) {
      try {
        console.log(`[Transaction] Attempt ${attempt}/${maxRetries}`);
        
        const result = await this.client.invokeContract(
          contractName,
          method,
          params
        );
        
        if (result.success) {
          return result;
        }
        
        // 检查是否为临时错误，可重试
        if (this.isRetryableError(result.message)) {
          lastError = new Error(result.message);
          if (attempt < maxRetries) {
            await this.delay(retryDelay * attempt);
            continue;
          }
        }
        
        return result;
      } catch (error: any) {
        console.error(`[Transaction] Attempt ${attempt} failed: ${error.message}`);
        lastError = error;
        
        if (attempt < maxRetries && this.isRetryableError(error.message)) {
          await this.delay(retryDelay * attempt);
        }
      }
    }
    
    return {
      success: false,
      txId: '',
      message: `All ${maxRetries} attempts failed. Last error: ${lastError?.message}`
    };
  }
  
  /**
   * 查询交易状态
   * @param txId 交易ID
   */
  async getTransaction(txId: string): Promise<TransactionInfo | null> {
    try {
      const result = await this.client.queryContract(
        '_system_contract',
        'getTxByTxId',
        new Map([['tx_id', txId]])
      );
      
      if (result.success && result.data) {
        return JSON.parse(result.data);
      }
      return null;
    } catch (error) {
      console.error(`[Transaction] Failed to get tx: ${error}`);
      return null;
    }
  }
  
  /**
   * 等待交易确认
   * @param txId 交易ID
   * @param timeout 超时时间（毫秒）
   */
  async waitForConfirmation(
    txId: string, 
    timeout: number = 30000
  ): Promise<boolean> {
    const startTime = Date.now();
    
    while (Date.now() - startTime < timeout) {
      const txInfo = await this.getTransaction(txId);
      
      if (txInfo) {
        if (txInfo.status === 'SUCCESS') {
          return true;
        }
        if (txInfo.status === 'FAILED') {
          return false;
        }
      }
      
      await this.delay(500);
    }
    
    return false;
  }
  
  /**
   * 判断是否为可重试错误
   */
  private isRetryableError(message?: string): boolean {
    if (!message) return false;
    
    const retryablePatterns = [
      'timeout',
      'network',
      'connection',
      'service unavailable',
      'try again'
    ];
    
    return retryablePatterns.some(pattern => 
      message.toLowerCase().includes(pattern)
    );
  }
  
  /**
   * 延迟函数
   */
  private delay(ms: number): Promise<void> {
    return new Promise(resolve => setTimeout(resolve, ms));
  }
}

interface RetryOptions {
  maxRetries?: number;
  retryDelay?: number;
}

interface InvokeResult {
  success: boolean;
  txId: string;
  message?: string;
  data?: any;
}

interface TransactionInfo {
  txId: string;
  status: 'PENDING' | 'SUCCESS' | 'FAILED';
  blockHeight?: number;
  message?: string;
}

/**

  async setSession(userId: string, data: object, ttl: number = 86400): Promise<void> {

  async getSession(userId: string): Promise<object | null> {

   * 删除用户会话

  async addUserDID(userId: string, did: string): Promise<void> {
    const key = `user:${userId}:dids`;
    await this.client.sadd(key, did);
  }
  
  /**
   * 获取用户的所有DID
   */
  async getUserDIDs(userId: string): Promise<string[]> {
    const key = `user:${userId}:dids`;
    return await this.client.smembers(key);
  }
  
  /**
   * 删除用户的DID索引
   */
  async removeUserDID(userId: string, did: string): Promise<void> {

    await this.client.srem(key, did);

  async cacheQuery(key: string, data: object, ttl: number = 300): Promise<void> {

   * @param limit 限制次数

  async rateLimit(identifier: string, limit: number, window: number): Promise<boolean> {
    const key = `ratelimit:${identifier}:${Math.floor(Date.now() / 1000 / window)}`;
    const current = await this.client.incr(key);

    const result = await this.client.exists(key);
    return result === 1;
  }
  
  /**
   * 关闭连接
   */
  async close(): Promise<void> {
    await this.client.quit();
  }
}

type Identity struct {

    StatusSuspended = "suspended"
    StatusRevoked  = "revoked"
)

// RegisterIdentity 注册新的数字身份
func RegisterIdentity(did, didDocument string) (*chainmaker.Response, error) {
    // 1. 验证DID格式
    if !ValidateDID(did) {
        return chainmaker.ErrorResult(10001, "invalid DID format")
    }
    
    // 2. 检查是否已存在
    existing, err := GetIdentity(did)
    if err != nil {
        return nil, err
    }
    if existing != nil {
        return chainmaker.ErrorResult(10004, "identity already exists")
    }
    
    // 3. 获取调用者身份
    caller, err := chainmaker.GetCaller()
    if err != nil {
        return nil, err
    }
    
    // 4. 创建身份记录
    now := time.Now().Unix()
    identity := &Identity{
        DID:       did,
        Document:  didDocument,
        Owner:     caller,
        Status:    StatusActive,
        CreatedAt: now,
        UpdatedAt: now,
    }
    
    // 5. 保存到链上
    if err := SaveIdentity(did, identity); err != nil {
        return nil, err
    }
    
    // 6. 记录事件
    chainmaker.EmitEvent("IdentityRegistered", did)
    
    fmt.Printf("[Identity] Registered: %s\n", did)
    
    result, _ := json.Marshal(map[string]interface{}{
        "success": true,
        "did":     did,
    })
    return chainmaker.SuccessResult(result)
}

// VerifyIdentity 验证身份有效性
func VerifyIdentity(did string) (*chainmaker.Response, error) {

    if err != nil {
        return nil, err
    }
    if identity == nil {
        return chainmaker.ErrorResult(10003, "identity not found")
    }
    
    // 2. 检查状态
    isValid := identity.Status == StatusActive
    
    result, _ := json.Marshal(map[string]interface{}{
        "did":       did,
        "valid":     isValid,
        "verifiedAt": time.Now().Unix(),
    })

package storage

import (
    "encoding/json"
    
    "chainmaker.org/chainmaker/sdk-golang/chainmaker"
)

const (
    IdentityKeyPrefix = "identity:"
    HistoryKeyPrefix  = "history:"
)

// Identity 身份结构体（与identity包共享）
type Identity struct {
    DID       string `json:"did"`
    Document  string `json:"document"`
    Owner     string `json:"owner"`
    Status    string `json:"status"`
    CreatedAt int64  `json:"created_at"`
    UpdatedAt int64  `json:"updated_at"`
    RevokedAt int64  `json:"revoked_at,omitempty"`
}

// HistoryEntry 历史记录条目
type HistoryEntry struct {
    Timestamp int64  `json:"timestamp"`
    Operation string `json:"operation"`
    Data      string `json:"data"`
}

// GetIdentity 获取身份信息
func GetIdentity(did string) (*Identity, error) {
    key := IdentityKeyPrefix + did
    data, err := chainmaker.GetState(key)
    if err != nil {
        return nil, err
    }
    if data == "" {
        return nil, nil
    }
    
    var identity Identity
    if err := json.Unmarshal([]byte(data), &identity); err != nil {
        return nil, err
    }
    return &identity, nil
}

// SaveIdentity 保存身份信息
func SaveIdentity(did string, identity *Identity) error {
    key := IdentityKeyPrefix + did
    data, err := json.Marshal(identity)
    if err != nil {
        return err
    }
    
    if err := chainmaker.PutState(key, string(data)); err != nil {
        return err
    }
    
    // 保存历史记录
    return SaveHistory(did, "UPDATE", string(data))
}

// GetHistory 获取历史记录
func GetHistory(did string) ([]HistoryEntry, error) {
    key := HistoryKeyPrefix + did
    data, err := chainmaker.GetState(key)
    if err != nil {
        return nil, err
    }
    if data == "" {
        return []HistoryEntry{}, nil
    }
    
    var history []HistoryEntry
    if err := json.Unmarshal([]byte(data), &history); err != nil {
        return nil, err
    }
    return history, nil
}

// SaveHistory 保存历史记录
func SaveHistory(did, operation, data string) error {
    key := HistoryKeyPrefix + did
    
    // 获取现有历史
    history, _ := GetHistory(did)
    if history == nil {
        history = []HistoryEntry{}
    }
    
    // 添加新记录
    history = append(history, HistoryEntry{
        Timestamp: identity.CreatedAt,
        Operation: operation,
        Data:      data,
    })
    
    historyData, err := json.Marshal(history)
    if err != nil {
        return err
    }
    
    return chainmaker.PutState(key, string(historyData))
}

// backend/tests/mocks/chainmaker.mock.ts

/**
 * 长安链SDK模拟器
 * 用于单元测试，无需连接真实区块链网络
 */
export class ChainMakerMock {
  private storage: Map<string, any> = new Map();
  private transactions: Transaction[] = [];
  
  /**
   * 模拟合约调用
   */
  async invokeContract(params: {
    contractName: string;
    method: string;
    params: Map<string, string>;
  }): Promise<InvokeResult> {
    const txId = this.generateTxId();
    
    this.transactions.push({
      txId,
      contractName: params.contractName,
      method: params.method,
      timestamp: Date.now()
    });
    
    // 模拟处理逻辑
    const result = this.processMethod(
      params.method,
      Object.fromEntries(params.params)
    );
    
    return {
      success: true,
      txId,
      message: 'OK',
      data: result
    };
  }
  
  /**
   * 模拟合约查询
   */
  async queryContract(params: {
    contractName: string;
    method: string;
    params: Map<string, string>;
  }): Promise<QueryResult> {
    const data = this.processMethod(
      params.method,
      Object.fromEntries(params.params)
    );
    
    return {
      success: true,
      data
    };
  }
  
  /**
   * 处理合约方法调用
   */
  private processMethod(method: string, params: Record<string, string>): any {
    switch (method) {
      case 'RegisterIdentity':
        return this.mockRegister(params);
      case 'VerifyIdentity':
        return this.mockVerify(params.did);
      case 'QueryIdentity':
        return this.mockQuery(params.did);
      case 'UpdateIdentity':
        return this.mockUpdate(params.did, params.did_document);
      case 'RevokeIdentity':
        return this.mockRevoke(params.did);
      default:
        throw new Error(`Unknown method: ${method}`);
    }
  }
  
  /**
   * 模拟注册
   */
  private mockRegister(params: { did: string; did_document: string }): string {
    if (this.storage.has(params.did)) {
      throw new Error('Identity already exists');
    }
    this.storage.set(params.did, {
      did: params.did,
      document: params.did_document,
      status: 'active',
      createdAt: Date.now()
    });
    return 'OK';
  }
  
  /**
   * 模拟验证
   */
  private mockVerify(did: string): string {
    const identity = this.storage.get(did);
    if (!identity) {
      throw new Error('Identity not found');
    }
    return identity.status === 'active' ? 'true' : 'false';
  }
  
  /**
   * 模拟查询
   */
  private mockQuery(did: string): string | null {
    const identity = this.storage.get(did);
    return identity ? JSON.stringify(identity) : null;
  }
  
  /**
   * 模拟更新
   */
  private mockUpdate(did: string, did_document: string): string {
    const identity = this.storage.get(did);
    if (!identity) {
      throw new Error('Identity not found');
    }
    identity.document = did_document;
    identity.updatedAt = Date.now();
    return 'OK';
  }
  
  /**
   * 模拟吊销
   */
  private mockRevoke(did: string): string {
    const identity = this.storage.get(did);
    if (!identity) {
      throw new Error('Identity not found');
    }
    identity.status = 'revoked';
    identity.revokedAt = Date.now();
    return 'OK';
  }
  
  private generateTxId(): string {
    return 'tx_' + Math.random().toString(36).substr(2, 32);
  }
  
  /**
   * 清空存储（测试用）
   */
  clear(): void {
    this.storage.clear();
    this.transactions = [];
  }
  
  /**
   * 获取所有交易记录（测试用）
   */
  getTransactions(): Transaction[] {
    return [...this.transactions];
  }
}

interface Transaction {
  txId: string;
  contractName: string;
  method: string;
  timestamp: number;
}

describe('IdentityService', () => {

      expect(result.did).toBe(did);

      // 第一次注册
      await service.registerIdentity(did, didDocument);

      const didDocument = { id: did };
      
      await service.registerIdentity(did, didDocument);
      
      const transactions = mockChainMaker.getTransactions();
      expect(transactions.length).toBeGreaterThan(0);
      expect(transactions[0].method).toBe('RegisterIdentity');
    });

    it('should return true for active identity', async () => {
      const did = 'did:chainmaker:act123456789';
      const didDocument = { id: did };
      
      await service.registerIdentity(did, didDocument);
      const result = await service.verifyIdentity(did);
      
      expect(result).toBe(true);
    });
    
    it('should return false for revoked identity', async () => {
      const did = 'did:chainmaker:rev123456789';
      const didDocument = { id: did };
      
      await service.registerIdentity(did, didDocument);
      await service.revokeIdentity(did);
      
      const result = await service.verifyIdentity(did);
      expect(result).toBe(false);
    });
    
    it('should throw error for non-existent identity', async () => {
      const did = 'did:chainmaker:nonexistent123';
      
      await expect(service.verifyIdentity(did))
        .rejects.toThrow('Identity not found');
    });
  });
  
  describe('updateIdentity', () => {

      const originalDoc = { id: did, name: 'Original' };
      const updatedDoc = { id: did, name: 'Updated' };

    });
  });
});

## 九、常见问题与解决方案

### 9.1 环境配置问题

#### Q1: Docker安装失败

**问题描述：** 在Windows上安装Docker Desktop失败

**解决方案：**
1. 确保已启用Windows Subsystem for Linux (WSL2)
2. 以管理员身份运行PowerShell
3. 执行：`wsl --install`
4. 重启电脑后重新安装Docker Desktop

#### Q2: 长安链SDK连接超时

**问题描述：** 调用链上合约时提示连接超时

**解决方案：**
1. 检查链节点是否正常运行：`docker ps | grep chainmaker`
2. 确认端口未被占用：`netstat -tlnp | grep 12301`
3. 检查防火墙设置，放行相应端口
4. 确认SDK配置的节点地址正确

### 9.2 智能合约问题

#### Q3: 合约编译失败

**问题描述：** Go合约编译报错

**解决方案：**
```bash
# 检查Go版本（需要1.18+）
go version

# 清理缓存
go clean -cache

# 重新下载依赖
go mod download

# 如果是WASM编译问题，安装tinygo
# Ubuntu/Debian
wget https://github.com/tinygo-org/tinygo/releases/download/v0.28.1/tinygo_0.28.1_amd64.deb
sudo dpkg -i tinygo_0.28.1_amd64.deb

# macOS
brew install tinygo

# 编译WASM合约
tinygo build -target wasi -o identity_contract.wasm .

#### Q4: 合约部署失败

**问题描述：** 部署合约时提示权限不足

**解决方案：**
1. 确认使用的是管理员证书
2. 检查证书路径是否正确
3. 确认链配置文件中的admin权限设置

### 9.3 后端服务问题

#### Q5: Redis连接失败

**问题描述：** 后端服务无法连接Redis

**解决方案：**
```bash
# 检查Redis服务状态
sudo systemctl status redis-server

# 启动Redis
sudo systemctl start redis-server

# 测试连接
redis-cli ping
# 应返回: PONG

# 如果使用Docker
docker ps | grep redis
docker logs redis
```

#### Q6: JWT Token验证失败

**问题描述：** 正常登录后API仍返回未授权

**解决方案：**
1. 检查`config/index.ts`中的JWT_SECRET配置
2. 确认请求头格式正确：`Authorization: Bearer <token>`
3. 检查token是否过期
