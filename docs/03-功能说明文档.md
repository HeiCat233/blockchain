# 功能说明文档

## 1. 核心身份管理功能

### 1.1 工作台 (Dashboard)

- **功能描述**：系统首页，展示数据概览
- **主要内容**：
  - 总学生数统计
  - 在校/失效学生数统计
  - 电子学生证预览
  - 系统状态显示
  - 最近操作记录

### 1.2 学生管理 (Students)

- **功能描述**：管理所有学生身份
- **功能列表**：
  - 查看学生列表
  - 查看学生详情
  - 注销学生身份
  - 状态标签（在读/失效）

### 1.3 新生报到 (Register)

- **功能描述**：新学生身份注册
- **功能列表**：
  - 填写学生信息（DID、姓名、学院、专业）
  - 注册并上链存证
  - 表单验证
  - 操作提示

### 1.4 身份核验 (Verify)

- **功能描述**：验证学生身份有效性
- **功能列表**：
  - 输入 DID/学号核验
  - 显示核验结果（通过/不通过）
  - 展示核验详情
  - 区块链验证标识

### 1.5 链上记录 (Audit)

- **功能描述**：查看区块链操作记录
- **功能列表**：
  - 操作日志表格
  - 操作类型、时间、关联 DID
  - 区块链特性说明

## 2. 校园场景化服务（扩展功能）

### 2.1 消费管理 (Consumption)

- **功能描述**：管理学生消费记录
- **功能列表**：
  - **查看消费记录**：按时间倒序显示所有消费
  - **添加消费记录**：
    - 选择学生
    - 选择消费类型（餐饮/购物/交通/其他）
    - 输入金额、地点、描述
  - **数据统计**：总消费金额、消费笔数
  - **消费记录字段**：
    - 时间
    - 学生 DID
    - 消费类型
    - 金额
    - 地点
    - 描述

### 2.2 场馆预约 (Reservation)

- **功能描述**：校园场馆预约管理
- **功能列表**：
  - **场馆列表**：
    - 图书馆 📚
    - 体育馆 🏀
    - 计算机实验室 💻
    - 会议室 📋
  - **预约功能**：
    - 选择学生
    - 选择预约日期
    - 选择时间段
    - 输入参与人数
  - **预约记录**：
    - 查看所有预约
    - 显示预约状态（已确认/已取消）
    - 取消预约功能
  - **场馆信息**：
    - 场馆名称
    - 描述
    - 容纳人数

### 2.3 成就管理 (Achievement)

- **功能描述**：学生成就、证书管理
- **功能列表**：
  - **成就卡片展示**：
    - 成就类型图标
    - 成就名称
    - 颁发机构
    - 颁发日期
    - 成就描述
    - 区块链存证标识
  - **添加成就**：
    - 选择学生
    - 成就类型（奖学金/证书/竞赛/其他）
    - 成就名称
    - 颁发机构
    - 颁发日期
    - 描述
  - **成就类型**：
    - 奖学金 🎓
    - 证书 📜
    - 竞赛 🏅
    - 其他 ⭐

## 3. API 接口文档

### 3.1 身份管理 API

| 方法 | 路径 | 描述 |
|------|------|------|
| GET | /health | 健康检查 |
| GET | /api/identities | 获取所有身份列表 |
| POST | /api/identity/register | 注册身份 |
| GET | /api/identity/:did | 查询身份详情 |
| GET | /api/identity/:did/verify | 验证身份有效性 |
| PUT | /api/identity/:did | 更新身份 |
| DELETE | /api/identity/:did | 注销身份 |

### 3.2 消费管理 API

| 方法 | 路径 | 描述 |
|------|------|------|
| POST | /api/consumption | 添加消费记录 |
| GET | /api/consumption/:did | 获取指定学生的消费记录 |
| GET | /api/consumptions | 获取所有消费记录 |

### 3.3 场馆预约 API

| 方法 | 路径 | 描述 |
|------|------|------|
| GET | /api/venues | 获取场馆列表 |
| POST | /api/reservation | 添加预约 |
| GET | /api/reservation/:did | 获取指定学生的预约记录 |
| GET | /api/reservations | 获取所有预约记录 |
| PUT | /api/reservation/:id/cancel | 取消预约 |

### 3.4 成就管理 API

| 方法 | 路径 | 描述 |
|------|------|------|
| POST | /api/achievement | 添加成就 |
| GET | /api/achievement/:did | 获取指定学生的成就 |
| GET | /api/achievements | 获取所有成就 |

## 4. 数据模型

### 4.1 DID 身份文档

```javascript
{
  "id": "did:campus:2024001",
  "controller": "did:campus:admin",
  "publicKey": "STUDENT_PUBKEY_did:campus:2024001",
  "status": "active",
  "created": "2024-06-16T00:00:00.000Z",
  "updated": "2024-06-16T00:00:00.000Z"
}
```

### 4.2 消费记录

```javascript
{
  "id": "consumption_xxx",
  "did": "did:campus:2024001",
  "amount": 25.5,
  "type": "food",
  "location": "第一食堂",
  "description": "午餐",
  "timestamp": "2024-06-16T00:00:00.000Z",
  "status": "completed"
}
```

### 4.3 预约记录

```javascript
{
  "id": "reservation_xxx",
  "did": "did:campus:2024001",
  "venueId": "library",
  "venueName": "图书馆",
  "date": "2024-06-17",
  "timeSlot": "09:00-10:00",
  "attendees": 1,
  "status": "confirmed",
  "createdAt": "2024-06-16T00:00:00.000Z"

  "verified": true,

```
