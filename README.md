# 校园电子学生证管理系统

基于长安链的校园电子学生证管理系统，支持身份管理、消费记录、场馆预约、成就管理等功能。

## 🚀 快速开始

### 5 分钟启动

```powershell
# 1. 进入后端目录
cd backend

# 2. 安装依赖
npm install

# 3. 启动服务
node app.js
```

然后打开浏览器访问：http://localhost:3000

---

## ✨ 主要功能

- **身份管理**：新生报到、学生管理、身份核验
- **消费管理**：消费记录、消费统计
- **场馆预约**：场馆列表、预约管理
- **成就管理**：成就证书、上链存证
- **区块链**：长安链支持，数据不可篡改

---

## 🛠️ 技术栈

- **后端**：Node.js + Express.js
- **区块链**：长安链 (ChainMaker)
- **前端**：原生 HTML5 + CSS3 + JavaScript
- **智能合约**：Go + WASM
